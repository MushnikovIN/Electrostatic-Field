# Расчет и визуализация электростатического поля системы расщепленных проводов

Данный проект предназначен для математического моделирования, расчета и интерактивной визуализации электростатического поля системы параллельных цилиндрических проводников (включая расщепленные фазы) над идеально проводящей поверхностью земли.

## Физико-математическая основа

Расчет электрического поля основан на **методе эквивалентных зарядов** и **методе зеркальных отображений**. Влияние плоской заземленной поверхности ($y = 0$) учитывается путем введения фиктивных зеркальных зарядов противоположного знака, расположенных симметрично под землей.

Связь между потенциалами проводов $[U]$ и их линейными зарядами $[\tau]$ выражается через матрицу потенциальных коэффициентов $[\alpha]$:
$$[U] = [\alpha] \cdot [\tau]$$

Плотности линейных зарядов находятся путем решения системы линейных алгебраических уравнений (СЛАУ):
$$[\tau] = [\alpha]^{-1} \cdot [U]$$

### Основные расчетные формулы:
* **Потенциал в точке $(x, y)$:**
  $$\phi = \frac{1}{2\pi \varepsilon_0 \varepsilon} \sum_{i=1}^{N} \tau_i \ln\left(\frac{r_{image, i}}{r_{real, i}}\right)$$
* **Напряженность поля в точке $(x, y)$:**
  $$E = \sqrt{E_x^2 + E_y^2}$$
  Где $E_x$ и $E_y$ рассчитываются через суперпозицию полей от реальных проводов и их зеркальных двойников.

## Архитектура и структура проекта

Проект реализован на языке MATLAB с применением объектно-ориентированного программирования (ООП) и матричных вычислений.

### Состав файлов:
1. `main.m` — главный управляющий скрипт. Содержит конфигурацию исходных данных, инициализацию расчетов и организацию диалогового цикла с пользователем.
2. `Field.m` — основной ООП-класс (`handle`), хранящий параметры системы и инкапсулирующий методы расчета и графики.
3. `splited_to_elementary.m` — функция разложения расщепленных проводов на систему одиночных элементарных проводников.
4. `validate_wires.m` — функция первичной синтаксической и логической проверки входной структуры данных.
5. `validate_geometry.m` — функция проверки геометрической корректности (контроль пересечения проводов между собой и касания земли).

## Инструкция по запуску

1. Сконфигурируйте массив структур `Wires` в файле `main.m`.
2. Запустите скрипт `main.m`.
3. В появившемся графическом окне отобразится геометрия проводов, заземляющая плоскость и единицы измерения величин.
4. Следуйте текстовым инструкциям в командном окне для выбора режимов ввода и параметров расчета ($E$, $\phi$ или оба параметра).
